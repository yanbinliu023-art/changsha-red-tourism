# AGENTS.md

## Project

This is a Chinese course practice research website about the protection and development of red tourism resources in Changsha and surrounding areas.

## Tech stack

- Vite
- React
- TypeScript
- Leaflet for map
- Recharts for charts
- Static local data only

## Rules

- Do not invent real survey data. Use clearly marked demo data unless I provide real questionnaire results.
- Use Chinese UI text.
- Keep the website suitable for a university course project.
- Prioritize mobile-friendly layout.
- Every page should support the research theme: protection, development, satisfaction, and education value.
- Use accessible colors and readable typography.
- After changing code, run `npm run build`.

## Main pages

- Home
- Red tourism map
- Site detail cards
- Protection and development evaluation
- Satisfaction survey dashboard
- Route recommendations
- About research
